This is the Peer to peer Simulation Project. It uses the DEVS formalism and the CD++ toolkit.

The atomic and coupled models are not stand alone, they require the CD++ eclipse toolkit to work. For more  information on the CD++ toolkit, see the following link

CD++ Main Page - http://cell-devs.sce.carleton.ca/mediawiki/index.php/Main_Page

CD++ installation instructions - http://www.youtube.com/watch?v=SrumQNqifPU

Check the p2p/documentation folder for more information, and the release folder for a downloadable package of the P2PSimulator.
